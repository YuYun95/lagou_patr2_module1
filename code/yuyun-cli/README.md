### 第三方库
* commander: 命令行工具
* download-git-repo: 用来下载远程模板
* inquirer: 交互式命令行工具
* ora: 显示loading动画
* log-symbols: 显示出 √ 或 × 等的图标
* chalk: 终端的字体样式
* ejs: 模板引擎
